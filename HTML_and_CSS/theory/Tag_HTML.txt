* <!DOCTYPE> - задає версію HTML
* <head> - визначає інфо про документ 
* <header> - позначає розділ HTML-сторінки як заголовок
* <html> - вказує що почався HTML-документ

* <h> - заголовок (<h1>  -  <h6>)

* <button> - кнопка

* <p> - абзац
* <section> - секція 
* <div> - блок, об'єднує кілька елементів інтерфейсу в один контейнер
* <aside> - блок з доп інфо або SIDEBAR, дає змогу пошуковій системі кращє розуміти що це доп інфо

* <code> - вказує що вміст тега програмниц код
* <script> - додає на веб-сторінку JS скрипт
* <link> - служить для зв'язування HTML-документа з іншим файлом, наприклад з CSS.Містить два компоненти:
              -rel - 
              -href - вказати адресу файлу
* <a> - посилання. За замовчуванням має синій колір Містить дві частини:
      - href - пишемо адресу сайту куди ми хочемо перейти
      - контент

* <ol> - створює обгортку нумерованого списку, в середині якого знаходиться тег <li> 
  * <li> - (list item) визначає елемент  списку

* <ul> - обгортка ненумерованого списку


* <audio> - додає аудіо запис
* <img> - відображає графічне зображення. Містить дві частини:
          - src - шлях до картинки
          - alt - і-нфо на випадок якщо не завантажиться картинка
* <input> - поле вводу даних

* <dfn> - позначає термін в тексті
* <dialog> - стрворює діалогове вікно
* <big> - робить текст більшим
* <i> - виділяє текст курсивом
* <bdo> - змінює напрямок тексту
* <center> - вирівнює текст по центру
* <b> - робить текст жирним 
* <basefont> - задає шрифт за замовчуванням
* 


* <span> - визначає унікальний ідентифікатор частини тексту
