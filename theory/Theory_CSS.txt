1.Щоб підключити CSS стилі потрібно в HTML-файлі в тезі <head> записати 
<link>, де буде посилання на папку і файл.
Стилізація можу бути:
        - через теги
        -через класи

    *Стилізація по тегах
-Вказується тег який потрібно стилізувати, потім в {} вказуємо CSSC властивості які ми хочемо додати
li {}, h3{}

*Властивості:
- background-color - колір фону
- color - колір шрифта

Але стилізація по тегах не завжди доречна, тому що якщо потрібно окремо стилізувати якийсь тег(h3) то вміст одного тегу стилізувати не вийде. Стилізація по тегах охоплює стилізацію всіх тегів, а не одного конкретного


    *Стилізація по класам
Це спеціальна назва яку можна дати тегу і стилізувати його по конкретно новоотриманій назві
Щоб задати тегу клас потрібно:
    -вказати клас : <h3>
    - написати в самому тегові class:   <h3 class="назва класу">

І тепер по заданому класу стилізуємо тег:
          .project_title {
          font-size: 45px;
          color: brown ;
          }


*Властивості:
- font-size: - розмір шрифта

Перевага класу в тому, що можна ідивідуально стилізувати кожен HTML-тег

                                       
Шрифти для CSS можна взяти на сайті https://fonts.google.com/ 
  Вибравши потрібний шрифт ми копіюємо його посилання і вставляємо в HTML-файл в <head>. Виглядає тепер це ось так:

    <!-- Підключенн шрифтів -->
      <link rel="preconnect" href="https://fonts.googleapis.com">
      <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
      <link href="https: //fonts.googleapis.com/css2? родина= Монсеррат:wght@400;700 & display=swap" rel="stylesheet">
    <!-- Підключення файлу CSS -->
      <link rel="stylesheet" href="styles/styles.css">
      <title>Visit Card</title>
    </head>

Тепер ці шрифти можна викорстовувати. Щоб використати їх в CSS, на тій сторінці гугл де ми взяли посилання на шрифти, є інфо як їх використати:
    font-family: 'Montserrat', sans-serif;

Щоб шрифти використовувались до всього файлу, потрібно вставити їх в тег <body>:
          body {
          font-family: 'Montserrat', sans-serif;
          }






                          3. Відступи
  За відступи між елементами на екрані відповідає дві властивості:
        -Margin - використовується для зовнішніх відступів(між блоками)
        -Padding - відступи в середині блока

*Margin записується :
1)
  margin-top: 40px;
  margin-bottom: 50px;
  margin-left: ;
  margin-right: ;
2)
В скороченому синтаксисі за замовчуванням спочатку вказується відступ зверху, потім справа і т.д
  margin: 100px 90px 300px 60px;

3)
Якщо записується один показник, то це ознаає однакові відстані з усіх 4 сторін
margin: 50px;

4)
Перша цифра 100px відповідає за top i bottom, а друга 50px за left i right
margin: 100px 50px;

*Padding записується такимиж варіантами як і margin







          4. Розміщення вертикальних блоків Sidebar i Content 
Для цього використовується технологія FLEXBOX.
В нас є два контейнера Sidebar i Content і щоб запацював flexbox потрбіно контейнеру записати спеціальне право.

* display: - вказує як має відображатись певний елемент HTML
* display: flex; - Ставить два блоки в одну лінію. В даному випадку Sidebar справа, а головний контент зліва
* flex-direction: row-reverse; - перевертає лінію. Тепер Sidebar зліва, а контент справа


Щоб налаштувати пункти списку, потрібно обнулити відступи (padding: 0;), налаштовати їх самостійно і додати стиль

Пріорітети стилів

Пріорітет вилаштовується по селекторам. Різні селектори мають різну пріорітетність.
Стилі можна отримувати з різних джерел. Це можуть бути різні фацли із стилями, стилі можуть задаватись різними селекторами і впливати на одні і тіж елементи. Це може призвести до конфліктів стилів. Для цього була придумана система приоритетів

Низький пріоритет:
-*
-h1 
-p

Високий приоритет
Звернення через id
#id-name
